# yuuka-mcp-office-reader

`yuuka-mcp-office-reader` 是 YUUKA 的特制 Office Reader MCP Server，用来读取本机的 Office/文档文件（PDF/DOCX/XLSX/PPTX 等），通过 stdio 提供 MCP 工具。

## 直接运行（npx）

```bash
npx -y yuuka-mcp-office-reader
```

## 配到 YUUKA

```bash
yuuka mcp add office-reader npx -y yuuka-mcp-office-reader
```

## 发布（在这个目录执行）

```bash
npm publish
```
